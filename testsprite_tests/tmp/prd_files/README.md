# Turbo Elite Motors

Welcome to the Turbo Elite Motors WordPress project repository. 

## Overview
This repository contains the custom development files, themes, and specific plugins for the Turbo Elite Motors website. 

### Structure
- **Core Files & Third-Party Plugins:** Excluded from version control for security and cleanliness via `.gitignore`.
- **Custom Themes:** Located in `wp-content/themes`. This project primarily utilizes an `astra-child` theme for custom development and overrides.

## Setup Instructions
1. Clone the repository into your local server environment's `wp-content` directory (or map it following your preferred local setup, such as Local by Flywheel).
2. Install WordPress.
3. Install standard required plugins independently.
4. Activate the `astra-child` theme from the WordPress Appearance menu.

## Technologies Used
- WordPress
- Astra Theme (Child Theme Setup)
- CSS/JavaScript (Custom modifications including Dark Mode widgets and responsive elements)
- Git & GitHub for Version Control
